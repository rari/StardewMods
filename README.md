# StardewMods

A collection of mods for Stardew Valley using SMAPI.

## Mods

### Quick Loot

Quickly loot all items from chests and containers. Features configurable hotkey and auto-close option.

**Features:**
- Quick Loot: Press Tab or Left Stick to instantly grab all items from a chest
- Quick Stow: Press L to transfer all items from your inventory to an open chest (respects ConvenientInventory favorites)
- Auto-close menu after looting all items
- Full GMCM (Generic Mod Config Menu) integration
- i18n support

**Author:** JennaJuffuffles  
**Version:** 1.0.0  
**Minimum SMAPI Version:** 4.0.0

**Installation:**
1. Install [SMAPI](https://smapi.io/)
2. Download the latest release from the [QuickLoot folder](QuickLoot/)
3. Extract to your `Mods` folder
4. Run the game through SMAPI

**Configuration:**
Configure via Generic Mod Config Menu in-game, or edit `config.json` manually.
